# Restaurant Delivery Order Processing System

## Вариант: ИБ И32 П31 Д1031 Д10O3 Д2П2 Д2Б2 OР2 ОД2

### Расшифровка варианта:

- **ИБ**: Источник бесконечный (постоянный поток клиентов)
- **И32**: Источник №3 с равномерным распределением времени между заказами
- **П31**: Прибор №3 с экспоненциальным распределением времени обслуживания
- **Д1031**: Дисциплина постановки в буфер - по кольцу
- **Д10O3**: Дисциплина отказа - выбивание самого старого заказа при переполнении
- **Д2П2**: Дисциплина выбора прибора - первый свободный прибор
- **Д2Б2**: Дисциплина выбора из буфера - FIFO (первый пришел, первый обслужен)
- **OР2**: Отображение результатов - графики и статистика
- **ОД2**: Динамическое отображение - схема модели с текущим состоянием

### Бизнес-домен: Система обработки онлайн-заказов в ресторане доставки еды

#### Элементы системы:

1. **Источник (ИБ, И32)**: Поток клиентов, делающих заказы через мобильное приложение или сайт
2. **Заявка**: Заказ на доставку еды (состав блюда, адрес, время заказа)
3. **Прибор (П31)**: Повар или кухонная линия с экспоненциальным временем приготовления
4. **Буфер**: Список заказов, ожидающих приготовления (максимум 20 заказов)
5. **Диспетчер**: Система управления заказами ресторана

#### Дисциплины работы:

- **Постановка в буфер**: По кольцу - система ищет первое свободное место, начиная с указателя
- **Обработка отказа**: При переполнении буфера отменяется самый старый заказ
- **Выбор прибора**: Заказ направляется на первую свободную кухонную линию
- **Выбор из буфера**: Заказы готовятся в порядке поступления (FIFO)

### Режимы работы:

1. **Пошаговый режим**: Отслеживание работы системы на каждом шаге
2. **Автоматический режим**: Запуск симуляции на заданное время

### Методы симуляции:

1. **На основе времени**: Движение по временной шкале
2. **На основе особых событий**: Обработка событий из календаря событий

### Особенности реализации:

- Кольцевой буфер с указателями вставки и извлечения
- Статистика в реальном времени
- Визуализация состояния системы
- Генерация отчетов
- Гибкая настройка параметров
